Login Failures

OpenDSA's score server uses an SSL certificate signed by Virginia Tech which your browser does not trust by default. You must manually add a security exception for the score server. To do so, simply surf to https://opendsa.cs.vt.edu where you should be prompted with a screen saying that the SSL certificate is untrusted and asking whether you would like to add a security exception. Follow the directions on the page to add the exception. Afterwards you should be able to login.

If you see a message saying that "server cannot be contacted", or if you see an exercise that shows "__ out of __" above it.

Try the following:

The first step is to clear the browser cache, and then reload the page.

  • On Chrome, open the settings menu and click "History", then "Clear Browsing Data". Clear "Cookies and oher site and plug-in data".
  • On Firefox, open the settings menu, and click on the "History" icon. Then clear "Cache".
  • After clearing as appropriate for the browser, reload the page.
