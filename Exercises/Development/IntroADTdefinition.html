An abstract data type is:

"The realization of a data type as a software component."
  • "A concrete implementation of a class."
  • "A function that defines how an implementation should be written."
  • "A subset of a type."

An implementation of a data type is it's physical form.

An ADT is a model of a type. It does not define how the data type is implemented.

A type is a collection of values and therefore, a subset of an abstract data type, which defines both the type and a set of operations on the type.
