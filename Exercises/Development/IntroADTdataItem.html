A data item is:

"A member of a type"
  • "A realization of a type"
  • "An implementation of an ADT"
  • "An object instantiated from a class."

An ADT is the realization of a data type as a software component.

A data structure is the implementation of an ADT.

In OO programming, an ADT and it's implementation together form a class. An object is an instance of a class. A data item is a member of the class.

A data item is a piece of information whose value is drawn from a type.
