How many times in total will the body of the interior for loop execute? 

int sum = 0;
for (int i = 0; i < 3; i++)
{
    for (int j = 1; j <= 5; j++)
    {
        sum = sum + j;
    }
}
15 times
  • 45 times
  • 5 times
  • 3 times

Remember each time int i increases, the internal loop will have iterated 5 times
